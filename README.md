# Hamad Lab Portfolio

A light, product-lab style portfolio for Hamad Alismail's applied AI systems and weekly project builds.

## Current live demo assets

- AI Business Process Auditor demo v1
- WhatsApp Business Command Center demo v1
- Miso Creator Content OS project card
- LinkedIn Talent + Recruiter Radar project card
- WhatsApp-first resource/waitlist mockup

## Development

```bash
npm install
npm run dev
npm run build
```

## Status language

Use honest project labels:

- `Live demo` for built website/demo surfaces
- `Demo v1 built` for working mockups with sample data
- `Live build log` for active systems being documented
- `Planned` only when no artifact exists yet
- `Production` only after real users/data/integrations exist
